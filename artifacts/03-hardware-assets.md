# 하드웨어 자산 명세서 (Hardware Assets Specification)
**무인 오피스 IoT 시스템 물리적 장치 정의**

---

## 문서 목적

이 문서는 무인 오피스 시스템 구현을 위한 **물리적 하드웨어 장치**를 정의한다.

**핵심 원칙:**
- 도면 기반으로 실제 설치 가능한 장치 선정
- 산업용 등급 (Industrial Grade) 우선
- 유지보수 용이성 고려
- 확장성 및 교체 가능성 확보

**작성 기준:**
- 3가지 오피스 타입 (독립형, 프리미엄, 전층형) 모두 커버
- 각 Zone별 필요 장치 리스트 도출
- 장치별 기술 사양 명시

---

## 1. 오피스 타입별 공간 구조 분석

### 1.1. 독립형 오피스 (Independent Type)

**구조 특징:**
- 건물 내 여러 층에 분산 입주
- 층별로 여러 회사가 공간 분할
- 공용 공간 (복도, 라운지) 多

**대표 평면도 (강남점 예시):**
```
[1층 - Public Zone]
- 메인 출입구 (유리 자동문)
- 로비
- 엘리베이터 홀

[3층 - Mixed Zone]
- 복도 (Shared)
- Zone-1: ABC사 사무실 (Private, 30평, 8명)
- Zone-2: XYZ사 사무실 (Private, 20평, 5명)
- Zone-3: GHI사 사무실 (Private, 25평, 7명)
- 공용 회의실 A (Reservation, 10인실)
- 공용 회의실 B (Reservation, 6인실)
- 라운지 (Shared)
- 팬트리 (Shared)

[5층 - Mixed Zone]
- (3층과 유사 구조)
```

**총 공간 수 (지점당 평균):**
- Public Zone: 1개 (1층 출입구)
- Shared Zone: 5개 (층별 복도 2개 + 라운지 2개 + 팬트리 1개)
- Reservation Zone: 4개 (회의실 4개)
- Private Zone: 8개 (개별 사무실 8개)
- **총 18개 제어 지점**

---

### 1.2. 프리미엄 오피스 (Premium Type)

**구조 특징:**
- 한 층에 다양한 공간 유형 혼재
- 독립 사무실 + 공용 공간 믹스
- 회의실 규모 多 (소/중/대)

**대표 평면도 (홍대점 예시):**
```
[1층 - Public Zone]
- 메인 출입구 (자동문)
- 로비 + 접수 데스크
- 엘리베이터 홀

[5층 전체 - Premium Floor]
- 복도 (Shared)
- 오픈 라운지 (Shared, 50평)
- 독립 사무실 A (Private, 40평, 12명)
- 독립 사무실 B (Private, 30평, 8명)
- 독립 사무실 C (Private, 25평, 6명)
- 대형 회의실 (Reservation, 20인실)
- 중형 회의실 (Reservation, 10인실)
- 소형 회의실 x2 (Reservation, 4인실)
- 전화부스 x3 (Reservation, 1인실)
- 팬트리 (Shared)
- 화장실 앞 복도 (Shared)
```

**총 공간 수:**
- Public Zone: 1개
- Shared Zone: 4개 (복도 + 라운지 + 팬트리 + 화장실 복도)
- Reservation Zone: 7개 (회의실 4개 + 전화부스 3개)
- Private Zone: 3개 (독립 사무실 3개)
- **총 15개 제어 지점**

---

### 1.3. 전층형 오피스 (Full Floor Type)

**구조 특징:**
- 한 회사가 건물의 전체 층 독점 사용
- 엘리베이터 홀부터 Private Zone 시작
- 내부 공간은 회사 자유 설계

**대표 평면도 (판교점 예시):**
```
[1층 - Public Zone]
- 메인 출입구
- 로비

[5층 전체 - ABC사 전용]
- 엘리베이터 홀 진입문 (Private Zone 시작)
- 내부 복도 (Private, 회사 전용)
- 오픈 오피스 (Private, 100평, 30명)
- 임원실 (Private, 20평, 2명)
- 회의실 A (Private, 내부용, 예약 선택)
- 회의실 B (Private, 내부용)
- 회의실 C (Private, 내부용)
- 휴게실 (Private)
- 팬트리 (Private)
```

**총 공간 수:**
- Public Zone: 1개 (1층)
- Private Zone: 9개 (엘리베이터 홀 진입 + 내부 8개 구역)
- **총 10개 제어 지점**

## 2. Zone별 장치 매핑 (Device Mapping)

### 2.1. Public Zone (메인 출입구)

**공간 특성:**
- 24시간 출입 가능
- 가장 많은 사람 통과
- 보안 최우선

#### 필요 장치 리스트

**A. 출입 제어 장치**
```
1. 전기정 (Electric Lock)
   - 형태: EM Lock (전자석 잠금장치)
   - 이유: 유리 자동문에 적합
   - 수량: 1개 (양쪽 문 시 2개)
   - 잠금력: 300kg (약 3,000N)

2. 통합 단말기 (All-in-One Terminal)
   - 기능: RFID/NFC 리더 + QR 스캔 + 화면 + 스피커
   - 디스플레이: 7인치 터치스크린
   - 카메라: 2MP (QR 스캔용)
   - 통신: Ethernet (PoE), Wi-Fi 백업
   - 수량: 1개 (외부 진입용)

3. 퇴실 버튼 (Exit Button)
   - 형태: 무선 푸시 버튼 (적외선 센서 내장)
   - 위치: 내부 벽면
   - 수량: 1개

4. 도어 센서 (Door Sensor)
   - 형태: 자기 센서 (Magnetic Contact)
   - 기능: 문 열림/닫힘 감지
   - 수량: 1개
```

**B. 모니터링 장치**
```
5. 재실 감지 센서 (Occupancy Sensor)
   - 형태: mmWave 레이더 센서
   - 감지 범위: 10m 반경
   - 정확도: ±0.5m
   - 수량: 1개 (로비 천장)

6. 환경 센서 (IAQ Sensor)
   - 측정 항목: 온도, 습도, CO2, PM2.5, VOC
   - 통신: Zigbee 또는 LoRaWAN
   - 수량: 1개
```

**C. 환경 제어 장치**
```
7. 조명 제어 릴레이 (Lighting Relay)
   - 용량: 10A/220V
   - 채널: 2채널 (주 조명 + 보조 조명)
   - 수량: 1개

8. HVAC 제어 모듈 (선택)
   - 형태: 기존 에어컨 IR 제어 또는 RS485 연동
   - 수량: 1개
```

**D. 전원 및 백업**
```
9. PoE 스위치 (Power over Ethernet)
   - 포트: 4포트 (통합 단말기 + 예비)
   - 전력: 30W/포트 (PoE+ 지원)
   - 수량: 1개

10. UPS (무정전 전원 장치)
    - 용량: 500VA
    - 백업 시간: 4시간 (전기정 + 단말기)
    - 수량: 1개
```

**장치 수량 요약:**
- 전기정: 1개
- 통합 단말기: 1개
- 퇴실 버튼: 1개
- 도어 센서: 1개
- 재실 센서: 1개
- 환경 센서: 1개
- 조명 릴레이: 1개
- PoE 스위치: 1개
- UPS: 1개
- **총 9개 장치**

### 2.2. Shared Zone (복도/라운지/팬트리)

**공간 특성:**
- 입주사 전체 자유 이용
- 조명 자동 제어 중요
- 재실 감지 필요

#### 필요 장치 리스트 (복도 기준)

**A. 출입 제어 장치**
```
1. 전기정 (Electric Lock)
   - 형태: Electric Strike (전기식 스트라이크)
   - 이유: 일반 목재/철재 문에 적합
   - 수량: 1개

2. 단말기 (Compact Terminal)
   - 기능: RFID/NFC 리더 + LED + 스피커
   - 디스플레이: 없음 (단순 인증만)
   - 수량: 1개

3. 퇴실 버튼
   - 수량: 1개

4. 도어 센서
   - 수량: 1개
```

**B. 모니터링 장치**
```
5. 재실 감지 센서
   - 형태: mmWave 레이더
   - 수량: 2개 (복도 양 끝)
```

**C. 환경 제어 장치**
```
6. 조명 제어 릴레이
   - 채널: 4채널 (구역별 조명 분할)
   - 수량: 1개
```

**장치 수량 요약 (복도 1개당):**
- 전기정: 1개
- 단말기: 1개
- 퇴실 버튼: 1개
- 도어 센서: 1개
- 재실 센서: 2개
- 조명 릴레이: 1개
- **총 7개 장치**

## 3. 물리적 장치 스펙 상세

### 3.1. 전기정 (Electric Lock) 선정 기준

#### A. EM Lock (전자석 잠금장치)

**적용 대상:**
- 유리 자동문 (Public Zone 메인 출입구)
- 양쪽 개폐 가능한 문
- 비상 탈출이 중요한 위치

**기술 사양:**
```
잠금력: 280~350kg (2,800~3,500N)
작동 전압: DC 12V
소비 전력: 약 5W (잠금 시), 0W (개방 시)
내구성: 100만회 이상
설치 위치: 문틀 상단
장점:
  - 정전 시 자동 개방 (Fail-Safe)
  - 화재 시 비상 탈출 용이
  - 유리문에 구멍 뚫기 불필요
단점:
  - 문이 완전히 닫혀야 작동
  - 전력 소모 지속 (잠금 시)

추천 모델:
  - Dorma EM 500 (350kg)
  - Assa Abloy EM3000 (300kg)
```

---

#### B. Electric Strike (전기식 스트라이크)

**적용 대상:**
- 일반 목재/철재 문 (복도, 회의실)
- 단방향 출입 (밖에서 안으로만 제어)
- 내부에서는 자유 퇴실

**기술 사양:**
```
잠금력: 500~800kg
작동 전압: DC 12V
소비 전력: 약 0.5W (대기), 8W (개방 시)
내구성: 50만회 이상
설치 위치: 문틀 중앙 (도어락 위치)
장점:
  - 기존 문에 설치 용이
  - 전력 소모 낮음 (대기 시)
  - 내부 자유 퇴실 (퇴실 버튼 불필요)
단점:
  - 문틀 가공 필요
  - Fail-Secure (정전 시 잠김 유지, 별도 배터리 필요)

추천 모델:
  - HES 1006 (800kg)
  - Trimec ES100 (500kg)
```

---

#### C. Electric Deadbolt (전기식 데드볼트)

**적용 대상:**
- 보안 중요 공간 (회의실, 개별 사무실)
- 양방향 제어 (안팎 모두 인증 필요)
- 높은 잠금력 필요

**기술 사양:**
```
잠금력: 1,000kg 이상
작동 전압: DC 12V
소비 전력: 약 0.3W (대기), 15W (동작 시 순간)
내구성: 100만회 이상
설치 위치: 문 중앙 (기존 도어락 교체)
장점:
  - 높은 보안성 (물리적 잠금)
  - 전력 소모 매우 낮음
  - 양방향 제어 가능
단점:
  - 문 두께 40mm 이상 필요
  - 설치 복잡 (문 가공)
  - 비상 탈출 시 퇴실 버튼 필수

추천 모델:
  - Yale YDM4109 (전기식)
  - Samsung SHP-DP728 (디지털 도어락, 원격 제어 가능)
```

## 5. 배선 및 전원 설계

### 5.1. 전기정 배선 (Door Lock Wiring)

**EM Lock (전자석) 배선:**
```
[SMPS 12V] ─┬─ (+) ─→ [EM Lock]
             │
             └─ (-) ─→ [EM Lock] ─→ [Relay NO] ─→ GND

※ Relay: 게이트웨이 또는 단말기에서 제어
※ 정전 시: Relay 개방 → EM Lock 전원 차단 → 문 열림 (Fail-Safe)
```

**Electric Strike/Deadbolt 배선:**
```
[SMPS 12V] ─┬─ (+) ─→ [Deadbolt]
             │
             └─ (-) ─→ [Deadbolt] ─→ [Relay NC] ─→ GND

※ Relay: 평상시 닫힘 (NC), 개방 신호 시 열림
※ 정전 시: Relay 잠금 유지 → UPS 배터리 사용
```

---

### 5.2. 센서 배선

**유선 센서 (RS485):**
```
[게이트웨이 RS485 A/B] ─── Cat6 ─── [센서 1] ─── [센서 2] ─── [센서 3]
                                      (Daisy Chain, 최대 32개)
```

**무선 센서 (Zigbee):**
```
[게이트웨이 Coordinator] ~~~무선~~~ [센서 1]
                                    [센서 2]
                                    [센서 3]
※ 배선 불필요, 배터리 또는 USB 전원
```

  ├─ [UPS 500VA] ─┬─ [게이트웨이]
  │               ├─ [코어 스위치]

## 9. 기술 선정 근거 요약

### 9.1. 전기정 선정

| 구분 | 선정 이유 |
|------|----------|
| EM Lock | Fail-Safe (화재 시 자동 개방), 유리문 적합 |
| Electric Strike | 기존 문 활용, 내부 자유 퇴실 |
| Electric Deadbolt | 높은 보안성, 양방향 제어 |

---

### 9.2. 센서 선정

| 센서 | 기존 대안 | 선정 이유 |
|------|----------|----------|
| mmWave | PIR (적외선) | 정적 재실 감지, 정확도 10배, 오작동 없음 |
| IAQ | 단일 센서 | 다중 측정 통합, 산업용 등급, 장기 안정성 |

---

### 9.3. 게이트웨이 선정

**산업용 등급 이유:**
- 24시간 무중단 운영 필요
- 극한 온도 대응 (-20°C ~ 70°C)
- 다중 프로토콜 지원 (Zigbee, RS485, Modbus)
- Edge Computing (로컬 자율 동작)
