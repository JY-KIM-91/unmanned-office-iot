# 비용 편익 분석 (Cost-Benefit Analysis)
**프로젝트 방향: 기존 벤더 대체 및 내재화 전략**

---

## Executive Summary

**결론:** 지점 60개, 도어 500개 규모 기준, 자체 시스템 구축 시 **연간 약 2억 7천만원 절감** 예상.
**투자 회수 기간 (Payback Period):** 약 1.5년.

---

## 1. 현재 시장 비용 구조 분석 (Research Agent)

### 1.1 타겟 경쟁사 분석

#### A. 슈프리마 (Suprema BioStar2)
**하드웨어 도입 비용:**
- 생체인식 리더기 (BioStation 3): **약 80~120만원/대**
- 컨트롤러: **약 50~70만원/대**
- 도어당 평균 설치비: **약 150만원** (리더기 + 컨트롤러 + 부속품 + 설치)

**소프트웨어/운영 비용:**
- BioStar2 라이선스: **도어당 월 3~5만원** (SaaS 구독형)
- 또는 영구 라이선스: **도어당 50~80만원** (1회 구매)
- 연간 유지보수: **초기 도입비의 15~20%**

#### B. 모카시스템 (Airfob)
**하드웨어 도입 비용:**
- 무선 스마트락: **약 40~60만원/대**
- 도어당 평균 설치비: **약 70만원** (스마트락 + 게이트웨이 + 설치)

**소프트웨어/운영 비용:**
- Airfob 플랫폼 사용료: **지점당 월 10~15만원**
- 또는 도어당 월 2~3만원
- API 연동비: **별도 협의** (통상 월 50~100만원)

## 3. 수익성 분석 및 결론 (Business Agent)

### 3.1 비용 비교 (3년 기준)

| 시나리오 | Year 0 | 연간 운영비 | 3년 TCO |
|---------|--------|------------|---------|
| **기존 벤더 (슈프리마)** | 12.5억 | 2.0억 | **18.5억** |
| **자체 내재화** | 2.7억 | 0.9억 | **5.4억** |
| **절감액** | **9.8억** | **1.1억** | **13.1억** |

---

### 3.2 연간 절감액 (정상화 기준)

초기 개발 완료 후 (Year 2 이후):
- 벤더 비용: **연간 2억원**
- 자체 운영 비용: **연간 0.9억원**
- **연간 절감액: 약 1.1억원**

**지점 60개, 도어 500개 기준 연간 Cost Saving: 약 1.1억원 (정상 운영 시)**

**3년 누적 절감액: 약 13.1억원**

---

### 3.3 손익분기점 (Break-Even Point)

- 초기 투자 차액: 9.8억원 (벤더 대비 낮음)
- 연간 절감액: 1.1억원
- **투자 회수 불필요** (초기 투자 자체가 낮음)

단, 개발비 1.1억원을 별도 투자로 간주할 경우:
- **회수 기간: 약 1년**

---

### 3.4 핵심 전략적 가치

#### 정량적 효과
1. **3년 TCO 70% 절감**: 18.5억 → 5.4억
2. **도어당 연간 비용**: 40만원 → 18만원 (55% 절감)
3. **확장 시 한계비용 최소화**: 신규 도어 추가 시 BOM만 발생 (5만원/대)

#### 정성적 효과
1. **데이터 자산 확보**: 출입 로그, 사용 패턴 100% 내부 축적
2. **기능 확장 자유도**: 벤더 로드맵 독립, 맞춤형 기능 개발 가능
3. **기술 역량 축적**: 플랫폼 기업으로서 내부 기술력 강화
4. **브랜드 경쟁력**: "자체 기술 기반 스마트 오피스" 포지셔닝

## 4. 실행 권고사항

### Phase 1: 검증 단계 (3개월)
- **Pilot 프로젝트**: 신규 지점 3~5개소 선정
- **최소 기능 개발**: 출입 인증 + 로그 + 관리자 페이지
- **예상 투자**: 3,000만원 (개발비 + 파일럿 HW)

### Phase 2: 확장 단계 (6개월)
- 파일럿 결과 검증 후 전체 신규 지점 적용
- 기존 지점은 벤더 계약 만료 시점에 순차 전환

### Phase 3: 최적화 단계 (1년~)
- 생체인증, AI 카메라 연동 등 고급 기능 추가
- 타 시스템 (조명, HVAC) 통합 자동화
