# Founder Context

## 1. 공간 IoT 및 보안 운영 설계 경험

직방 가드 및 직방 라운지 프로젝트를 통해
공간 기반 IoT 서비스의 설계, 구축, 운영 구조를 경험.

### 경험 범위
- Zigbee 기반 세대 내 IoT 보안 시스템 설계
- CCTV (PoE / WiFi) 설치 정책 및 운영 가이드 수립
- NVR/NAT 설정 및 네트워크 구조 이해
- AWS IoT Core 기반 API 전환 설계
- MQTT 기반 이벤트 구조 설계
- VPN / VPC 기반 원격 모니터링 설계
- 건물 단위 전기/네트워크 도면 기반 인프라 설계

---

## 2. 기술보다 운영을 이해하는 PM

프로젝트 경험을 통해 확인한 핵심 인사이트:

- 무인화는 기술 프로젝트가 아니라 운영 구조 재설계 프로젝트
- 완전 자동화는 리스크가 높으며 단계적 접근이 필요
- 장애 대응 설계가 없는 자동화는 실패한다
- 하드웨어, 네트워크, 클라우드, 운영이 분리되면 지속 불가

---

## 3. 본 프로젝트에의 적용 가능성

- 공유 오피스 내 IoT 내재화 전략 수립 가능
- 파일럿 단위 자동화 설계 및 확장 로드맵 설계 가능
- 개발자 중심 구조가 아닌 PM 중심 통합 설계 가능
